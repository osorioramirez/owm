## React & Redux & Redux-Saga Example

This example is based on [React Starter Kit](https://www.reactstarterkit.com) boilerplate.

### Installation

  * Clone repository
  * ```npm install```

### Execution (DEV)

  * ```npm start```

### Execution (PROD)

  * ```npm run build```
  * ```cd build```
  * ```npm install```
  * ```npm start```
